# bufete-web

Sitio web del bufete de abogados.

## Descripción

Plataforma web para la gestión y presentación de servicios legales del bufete.

## Características

- Presentación de áreas de práctica legal
- Formulario de contacto para clientes
- Información del equipo de abogados
- Blog de noticias jurídicas

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/jcmartinez0421-cell/bufete-web.git

# Entrar a la carpeta
cd bufete-web
```
